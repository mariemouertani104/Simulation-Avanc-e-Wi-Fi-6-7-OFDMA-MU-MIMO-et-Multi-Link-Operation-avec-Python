# Simulation Avancée Wi‑Fi 6 & 7 : OFDMA, MU‑MIMO et Multi‑Link Operation avec Python

## Description

Ce projet est une **simulation avancée de Wi‑Fi 6 (802.11ax) et Wi‑Fi 7 (802.11be)** réalisée en **Python** sur Google Colab.  
L'objectif est de **visualiser et expérimenter les technologies clés** :

- **OFDMA (Orthogonal Frequency Division Multiple Access)** : allocation dynamique de Resource Units (RUs) aux stations.
- **MU-MIMO (Multi-User Multiple Input Multiple Output)** : transmission simultanée à plusieurs stations via antennes multiples.
- **MLO (Multi-Link Operation, Wi‑Fi 7)** : transmission sur plusieurs bandes (ex. 5 GHz + 6 GHz) pour augmenter le débit.

Ce projet est **pédagogique et fonctionnel**, permettant d’explorer les concepts Wi‑Fi avancés sans matériel réel.

---

## Fonctionnalités

1. **Allocation OFDMA**
   - Division de la bande passante en RUs
   - Attribution automatique des RUs aux stations selon le gain de canal

2. **Simulation MU-MIMO**
   - Vecteurs de beamforming simulés
   - Débit calculé avec effet beamforming

3. **Multi-Link Operation (Wi-Fi 7)**
   - Deux bandes simulées (5 GHz + 6 GHz)
   - Somme des débits pour débit total MLO

4. **Visualisation professionnelle**
   - Histogrammes du débit par station
   - Comparaison multi-bande
   - Heatmap des allocations de RUs

---
## Concepts pédagogiques
- OFDMA : permet à plusieurs stations d’utiliser différentes portions de la bande simultanément.
- MU-MIMO : améliore le débit total en envoyant des signaux distincts à plusieurs stations via antennes multiples.
- MLO : agrégation multi-bande pour augmenter le débit et réduire la latence, fonctionnalité clé de Wi‑Fi 7.
